Wprowadzenie

Pakiet ten pozwala w na dołożenie edycji właściwości obiektów do dowolnej aplikacji.

Główna klasą jest PropertyGrid, który pozwala w wizualny sposób wyedytować właściwości dowolnego obiektu implementującego interface IPropertyProvider.

Sposób użycia

  • Do UI aplikacji dodajemy obiekt PropertyGrid
  • W klasie Obiektu którego właściwości chcemy edytować implementujemy interface IPropertyProvider. Można też użyć klasy bazowej BasicPropertyProvider.
  • Aby wyedytwać właściwości obiektu wstawiamy go do property grid za pomocą funkcji setModule()

Rozszerzenie pakietu

W celu dołożenia obsługi własnych typów property należy:

  • Utworzyć nowy typ property rozszerzący interface IProperty
  • Utworzyć faktory dla property edytorów implementujące interface: IPropertyEditorFactory. Można w tym celu nadpisać klasę PropertyEditorFactory jeżeli chcemy zachować domyśle edytory i tylko dodać nowe własne.
  • Zaimplementować nowy typ edytora implementujący interface IPropertyEditor. Jeżeli chcemy dołożyć edytor oparty o dialogbox to można rozszerzyć klasę DlgPropertyEditor
